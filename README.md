# World Simulator

This repository contains a 2D turn-based world simulator developed as part of the **Programowanie Obiektowe** course for computer science students.

The simulator uses a graphical interface built with **Java Swing**. Users can interact with the world through buttons, menus, and keyboard controls.

## Features

- Graphical representation of the world using **Swing** components
- Turn-based simulation
- Controllable human character using keyboard arrows
- Special ability for the human character
- Save and load the world state to/from a file
- Ability to add organisms to the world by clicking on empty tiles
- Abstract world structure with two implementations:
  - Grid-based gameplay
  - Hexagonal board gameplay
